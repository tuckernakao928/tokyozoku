## 1. プロジェクト定義

- サイト名：週末東京族
- ドメイン：tokyozoku.com
- キャッチコピー：「地方に住む。東京を使う。」
- コンセプト：平日は地方で暮らし、週末は東京で過ごす
  2拠点ライフスタイルの提案メディア
- ポジション：「完全移住」でも「東京在住」でもない
  第三の選択肢。「逆参勤交代」という新しい生き方

## 3. 編集方針

### このサイトは「ライフスタイル提案メディア」である
「週末東京族」という新しい生き方を具体的に提示し、
読者が自分の生活にイメージを重ねられるコンテンツを作る。

### コンテンツ設計ルール
- 「週末東京族」というコミュニティの視点で書く
- 体験またはシミュレーションを必ず入れる
- 抽象論禁止。必ず具体例・数字を入れる
- 内部リンクを意識した構成にする

## 4. コンテンツ計画

### 初期優先5本
1. 移住コストシミュレーター（最重要フック）
2. 東京に通える移住先ランキング
3. 移動を快適にするアイテム特集
4. 東京での宿泊戦略ガイド
5. サブスク・拠点サービスまとめ

### フェーズ設計
- Phase 1：検索流入獲得（移住先ランキング・コスト比較）
- Phase 2：ファン化（移動グッズ・週末ルーティン・手土産）
- Phase 3：コミュニティ化（SNS連動・ユーザー参加型）
