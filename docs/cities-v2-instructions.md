## 1. 物価表示ロジックの修正

SimulatorClient.tsx の物価表示部分を以下のように修正する。

### 修正前
```tsx
・物価：東京より約{(100 - city.priceIndex).toFixed(1)}%安い
```

### 修正後
```tsx
・物価：{
  city.priceIndex < 99
    ? `東京より約${(100 - city.priceIndex).toFixed(1)}%安い`
    : city.priceIndex > 101
    ? `東京より約${(city.priceIndex - 100).toFixed(1)}%高い`
    : `東京とほぼ同じ`
}
```

## 3. 注意事項

- 既存26都市のデータは変更しないこと
- CITIES配列の末尾に追加すること
- 物価指数100の横須賀市は「東京とほぼ同じ」と表示される
- 物価指数101の鎌倉市・茅ヶ崎市は「東京より約1%高い」と表示される
- 補助金nullの都市（鎌倉・茅ヶ崎・横須賀）は「要確認」と表示されること
- ビルド後に動作確認してからデプロイすること
