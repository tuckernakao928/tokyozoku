## 0. 実装の全体方針

- 現行のシミュレーター（v1）を v2 に全面リニューアルする
- 画面を「STEP1：条件入力」→「STEP2：移住先を選ぶ」→「STEP3：結果」の3ステップに分離する
- 計算ロジックのバグを修正する
- GA4トラッキングを追加する
- LINE登録ボタンを結果画面に追加する（URLはプレースホルダー）

## 2. SimulatorClient.tsx の全面リニューアル

### 2-1. 画面構成

3ステップで構成する。

```
STEP1：条件入力
  - 月給スライダー（20万〜100万、デフォルト33万）
  - 現在の東京家賃スライダー（5万〜25万、デフォルト7万）
  - 東京への頻度トグル（月1回・月2回・月4回・週2回）
  - 宿泊泊数トグル（日帰り・1泊・2泊・3泊以上）
  - 宿泊費スライダー（宿泊泊数が1以上の場合のみ表示）
  - 「移住先を選ぶ」セクション：4つのボタン
    ・💰 コスパで選ぶ
    ・🚄 移動時間で選ぶ
    ・🏔 レジャーで選ぶ
    ・✨ オススメから選ぶ

STEP2：移住先を選ぶ（ボタンに応じて内容が変わる）
  ← 条件入力に戻るボタン

  【コスパで選ぶ】
    フィルター3つ（セレクトボックス）：
    - 補助金：すべて / 補助金あり
    - 所要時間：すべて / 1時間以内 / 1.5時間以内 / 2時間以内
    - 家賃上限：すべて / 〜5万円 / 〜7万円 / 〜10万円
    フィルター適用後の都市をnetMonthly降順でランキング表示
    各行：順位・都市名・家賃・所要時間・月間収支差額
    都市タップ → STEP3へ

  【移動時間で選ぶ】
    所要時間の上限を3つのボタンで選ぶ：
    - 1時間以内
    - 1.5時間以内
    - 2時間以内
    選択後、条件に合う都市を所要時間昇順で一覧表示
    各行：都市名・所要時間・家賃・月間収支差額
    都市タップ → STEP3へ

  【レジャーで選ぶ】
    タグを5つ表示（複数選択可）：
    - 🏔 山・自然
    - 🎿 スキー
    - 🌊 海・釣り
    - 🏯 観光地
    - 🛣 道の駅
    選択したタグを含む都市を一覧表示
    都市タップ → STEP3へ

  【オススメから選ぶ】
    4つのカテゴリーカードを表示：
    - 💰 補助金が手厚い → 大月市・秩父市・安中市
    - 🏠 とにかく家賃が安い → 安中市・大月市・銚子市
    - 🚄 東京から近いのに安い → 小山市・大月市・秩父市
    - 🍜 何でも美味い → 仙台市・浜松市・甲府市
    都市タップ → STEP3へ

STEP3：結果
  ← 移住先選択に戻るボタン

  ① メインヒーロー
    - 都市名
    - メインコピー：「年間 ¥XXX,XXX の余裕が生まれます」（マイナスの場合は「年間 ¥XXX,XXX のコスト増になります」）
    - サブコピー：「東京でのお小遣いとして使える金額です」（プラスの場合のみ）

  ② 4つのメトリクスカード
    - 年間交通費（赤）
      説明文：「週末東京へ行くための交通費」
      内訳：「月X回×往復¥X,XXX×12ヶ月」
    - 家賃年間差額（緑）
      説明文：「東京家賃との年間差額」
    - 移住補助金（緑、nullの場合は「要確認」）
    - 月あたり換算（プラス緑・マイナス赤）

  ③ 月額内訳（修正済み計算ロジックで表示）
    - 月給
    - 東京家賃（現在）：マイナス赤
    - 移住先家賃：緑で表示
    - 月の交通費（東京往復）：マイナス赤
    - 月の宿泊費（宿泊あり時のみ）：マイナス赤
    - 月間収支差額（合計）：netMonthlyを使用（プラス緑・マイナス赤）
    ※ 「月間収支差額」はrentSavingMonthlyではなくnetMonthlyを使うこと

  ④ 家賃比較表
    |  | 東京（現在） | ○○市 |
    | 月額家賃 | ¥XX,XXX | ¥XX,XXX |
    | 年間家賃 | ¥XXX,XXX | ¥XXX,XXX |
    | 差額 | - | ¥XXX,XXXお得 |

  ⑤ 都市概要
    - 都市名について
    - overviewテキスト
    - overviewTagsをタグ表示

  ⑥ その他データ
    - 物価：東京より約X%安い
    - レジャー充実度
    - 人気度（星表示）
    - 所要時間

  ⑦ LINE登録ボタン
    「移住情報をLINEで受け取る」
    href="https://lin.ee/XXXXXXXX"（プレースホルダー、後で差し替え）
    緑色ボタン

  ⑧ アフィリエイトリンク（現行と同じ4つ）

  ⑨ Xシェアボタン（現行と同じ）
```
